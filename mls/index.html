Machine Learning for Sciences (MLS)

Raghavendra Selvan, University of Copenhagen (UCPH)

Welcome to the interactive laboratory for the MLS course. These tools are designed to provide intuition about fundamental Machine Learning concepts.

  • Supervised Learning Linear Classifier Try out the influence of different parameters of the linear classifier to obtain good performance.
  • Supervised Learning Decision Trees Interactively split Temperature and Rainfall space to understand how CART algorithms minimize impurity.
  • Unsupervised Learning Representation Learning Interactively learn a representation space for high-dimensional objects like images.
  • Basics Curse of Dimensionality Interactively try out the impact of distances in high-dimensions.
  • Basics Backpropagation Interactively check out how the gradients are computed using chain rule in computational graphs.
  • Coming Soon Clustering Try out k-means clustering.
